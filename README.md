# Takanashi-Hoshino
Hello Sensei ~~

## Description
**Takanashi-Hoshino** is a Discord bot built with Node.js and Discord.js. It offers various features to help manage and enhance your Discord server experience.

---

## Setting Up the `.env` File:

Create a `.env` file in the root directory of your project with the following content:

```bash dotenv
token=YOUR_TOKEN_HERE
prefix=CUSTOM_PREFIX # Example: $
mongo_url=YOUR_MONGO_URL
```
